# Quotables - Full Stack Application

Quotables allows users to share their favourite quotes with the world.

## Features

- Sign up for new users and sign in for old users.
- Form validation and JWT authentication implemented for security.
- Users can post a new quote with title, author and content fields.
- Delete a quote you have posted.
- Fully responsive.

## Upcoming feature

- Ability for users to like a quote that they enjoyed.

## Technologies used

- MERN stack (MongoDB, Express.js, React, Node.js).
- Jest for testing.
- Tailwind and DaisyUI for styling.
- Popular React libraries such as React Hook Form and React Icons.
- Vite.
